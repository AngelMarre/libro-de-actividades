II. SDL Reference
5. General
SDL_Init — Initializes SDL
SDL_InitSubSystem — Inicializa subsistemas
SDL_QuitSubSystem — Cierra un subsistema
SDL_Quit — Cierra SDL
SDL_WasInit — Comprueba que subsistemas están inicializados
6. Video
SDL_GetVideoSurface — devuelve un puntero a la superficie de la pantalla actual
SDL_GetVideoInfo — devuelve un puntero a la información sobre el hardware de video
SDL_ListModes — Devuelve un puntero a una matriz de las dimensiones de pantalla disponibles para el formato y flags de video dados.
SDL_VideoModeOK — Comprueba si un modo concreto de video está soportado.
SDL_SetVideoMode — Set up a video mode with the specified width, height and bits-per-pixel.
SDL_UpdateRect — Makes sure the given area is updated on the given screen.
SDL_UpdateRects — Makes sure the given list of rectangles is updated on the given screen.
SDL_Flip — Swaps screen buffers
SDL_SetColors — Sets a portion of the colormap for the given 8-bit surface.
SDL_SetPalette — Sets the colors in the palette of an 8-bit surface.
SDL_SetGamma — Sets the color gamma function for the display
SDL_GetGammaRamp — Gets the color gamma lookup tables for the display
SDL_SetGammaRamp — Sets the color gamma lookup tables for the display
SDL_MapRGB — Map a RGB color value to a pixel format.
SDL_MapRGBA — Map a RGBA color value to a pixel format.
SDL_GetRGB — Get RGB values from a pixel in the specified pixel format.
SDL_GetRGBA — Get RGBA values from a pixel in the specified pixel format.
SDL_CreateRGBSurface — Create an empty SDL_Surface
SDL_CreateRGBSurfaceFrom — Create an SDL_Surface from pixel data
SDL_FreeSurface — Frees (deletes) and SDL_Surface
SDL_LockSurface — Lock a surface for directly access.
SDL_UnlockSurface — Unlocks a previously locked surface.
SDL_LoadBMP — Carga un fichero BMD de Windows en una superficie (SDL_Surface).
SDL_SaveBMP — Save an SDL_Surface as a Windows BMP file.
SDL_SetColorKey — Sets the color key (transparent pixel) in a blittable surface and RLE acceleration.
SDL_SetAlpha — Adjust the alpha properties of a surface
SDL_SetClipRect — Sets the clipping rectangle for a surface.
SDL_ConvertSurface — Converts a surface to the same format as another surface.
SDL_BlitSurface — Esto realiza un rápido blit de la superficie origen a la supurficie destino.
SDL_FillRect — This function performs a fast fill of the given rectangle with some color
SDL_DisplayFormat — Convert a surface to the display format
SDL_DisplayFormatAlpha — Convert a surface to the display format
SDL_WarpMouse — Set the position of the mouse cursor.
SDL_CreateCursor — Creates a new mouse cursor.
SDL_FreeCursor — Frees a cursor created with SDL_CreateCursor.
SDL_SetCursor — Set the currently active mouse cursor.
SDL_GetCursor — Get the currently active mouse cursor.
SDL_ShowCursor — Toggle whether or not the cursor is shown on the screen.
SDL_GL_LoadLibrary — Specify an OpenGL library
SDL_GL_GetProcAddress — Get the address of a GL function
SDL_GL_GetAttribute — Get the value of a special SDL/OpenGL attribute
SDL_GL_SetAttribute — Set a special SDL/OpenGL attribute
SDL_GL_SwapBuffers — Swap OpenGL framebuffers/Update Display
SDL_CreateYUVOverlay — Create a YUV video overlay
SDL_LockYUVOverlay — Lock an overlay
SDL_UnlockYUVOverlay — Unlock an overlay
SDL_DisplayYUVOverlay — Blit the overlay to the display
SDL_FreeYUVOverlay — Free a YUV video overlay
SDL_GLattr — SDL GL Attributes
SDL_Rect — Defines a rectangular area
SDL_Color — Descripción de un color con formato independiente.
SDL_Palette — Color palette for 8-bit pixel formats
SDL_PixelFormat — Stores surface format information
SDL_Surface — Graphical Surface Structure
SDL_VideoInfo — Video Target information
SDL_Overlay — YUV video overlay
7. Manejo de ventanas
SDL_WM_SetCaption — Establece el título de la ventana y el nombre de su icono.
SDL_WM_GetCaption — Obtiene el título de la ventana y el nombre de su icono.
SDL_WM_SetIcon — Establece el icono de la ventana mostrada.
SDL_WM_IconifyWindow — Minimiza/Iconiza la ventana
SDL_WM_ToggleFullScreen — Intercambia entre modo ventana y modo a pantalla completa
SDL_WM_GrabInput — Controla (grabs) la entrada del ratón y del teclado.

9. Joystick
SDL_NumJoysticks — Cuenta el número de joysticks disponibles.
SDL_JoystickName — Obtiene el nombre del joystick.
SDL_JoystickOpen — Abre un joystick para su uso.
SDL_JoystickOpened — Determina si un joystick ha sido abierto
SDL_JoystickIndex — Obtiene el íncide de un SDL_Joystick.
SDL_JoystickNumAxes — Obtiene el número de ejes de un joystick
SDL_JoystickNumBalls — Obtiene el número de trackballs
SDL_JoystickNumHats — Obtiene el número de minijoysticks(hats)
SDL_JoystickNumButtons — Obtiene el número de botones del joystick
SDL_JoystickUpdate — Actualiza el estado de todos los joysticks
SDL_JoystickGetAxis — Obtiene el estado actual de un eje
SDL_JoystickGetHat — Obtiene el estado actual del minijoystick(hat) de un joystick
SDL_JoystickGetButton — Obtiene el estado actual de un determinado botón de un joystick
SDL_JoystickGetBall — Obtiene el movimiento relativo del trackball
SDL_JoystickClose — Cierra un joystick previamente abierto
10. Audio
SDL_AudioSpec — Audio Specification Structure
SDL_OpenAudio — Abre el dispositivo de audio con los parámetros deseados.
SDL_PauseAudio — para y continua la reproducción (pausa) del proceso callback de audio.
SDL_GetAudioStatus — Obtiene el estado del audio actual
SDL_LoadWAV — Load a WAVE file
SDL_FreeWAV — Libera datos WAV abiertos previamente
SDL_AudioCVT — Conversión de la estructura de Audio
SDL_BuildAudioCVT — Initializes a SDL_AudioCVT structure for conversion
SDL_ConvertAudio — Convert audio data to a desired audio format.
SDL_MixAudio — Mezcla datos de audio
SDL_LockAudio — Bloque la función callback
SDL_UnlockAudio — Desbloquea la función
SDL_CloseAudio — Cierra el proceso de audio y el dispositivo de audio.
11. CD-ROM
SDL_CDNumDrives — Devuelve el número de unidades de CD-ROM que hay en el sistema.
SDL_CDName — Devuelve el identificador del CD-ROM dependiendo del sistema en formato texto (human-readable).
SDL_CDOpen — Opens a CD-ROM drive for access.
SDL_CDStatus — devuelve el estado actual de la unidad dada.
SDL_CDPlay — Reproduce un CD
SDL_CDPlayTracks — Play the given CD track(s)
SDL_CDPause — Pausa un CDROM
SDL_CDResume — Restaura la reproducción (quita la pausa)
SDL_CDStop — Para la reproducción
SDL_CDEject — Expulsa el CDROM
SDL_CDClose — Cierra el manejador (handle) de la unidad de CD_ROM
SDL_CD — CDROM Drive Information
SDL_CDtrack — CD Track Information Structure
12. Programación Multi-Hilo
SDL_CreateThread — Crea un nuevo hilo que comparte las propiedades de su padre.
SDL_ThreadID — Get the 32-bit thread identifier for the current thread.
SDL_GetThreadID — Obtiene el ID de hilo SDL de un SDL_Thread
SDL_WaitThread — Espera a la finalización de un hilo de ejecución.
SDL_KillThread — Elimina sin esperar el hilo de ejecución.
SDL_CreateMutex — Crear un mutex
SDL_DestroyMutex — Destruye un mutex
SDL_mutexP — Cierra un mutex
SDL_mutexV — Abre un mutex
SDL_CreateSemaphore — Crea un nuevo semáforo y le asigna un valor de inicio.
SDL_DestroySemaphore — Destruye un semáforo creado con SDL_CreateSemaphore.
SDL_SemWait — Cierra un semáforo y suspende el hilo en caso de que el valor del semáforo sea cero.
SDL_SemTryWait — Intenta cerrar un semáforo pero no suspende el hilo de ejecución.
SDL_SemWaitTimeout — Cierra un semáforo, pero espera únicamente durante un tiempo máximo.
SDL_SemPost — Abre un semáforo.
SDL_SemValue — Devuelve el valor actual de un semáforo.
SDL_CreateCond — Crea una variable de condición
SDL_DestroyCond — Destruye una variable de condición
SDL_CondSignal — Reinicia un hilo en espera de una variable de condición
SDL_CondBroadcast — Reinicia todos los hilos que están esperando a una variable de condición
SDL_CondWait — Espera a una variable de condición
SDL_CondWaitTimeout — Espera a una variable de condición, con tiempo de expiración
13. Time
SDL_GetTicks — Toma el numero de milisegundos desde que se inicializazó la libreria SDL.
SDL_Delay — Espera un tiempo especificado en milesegundos para despues continuar con la ejecución del programa.
SDL_AddTimer — Añade un timer que espera un cierto tiempo para depues llamar a la función callback.
SDL_RemoveTimer — Elimina un timer que fué añadido con SDL_AddTimer.
SDL_SetTimer — Cambia el intervalo de tiempo de un timer en ejecución e incluso a la función callback que ejecuta.
