SDL_SemPost

Name

SDL_SemPost -- Abre un semáforo.

Synopsis

#include "SDL.h"
#include "SDL_thread.h"

int SDL_SemPost(SDL_sem *sem);

Descripción

SDL_SemPost abre el semáforo al que apunta sem e incrementa automáticamente el valor del semáforo. Los hilos que estaban bloqueados en este semáforo pueden ser planificados tras una llamada con éxito a esta función.

SDL_SemPost debería ser llamada tras el cierre de un semáforo mediante una llamada con éxito a SDL_SemWait, SDL_SemTryWait o SDL_SemWaitTimeout.

Valor Devuelto

Devuelve 0 en caso de éxito o -1 si se produjo algún (dejando el semáforo sin cambios).

Ejemplos

SDL_SemPost(my_sem);
