SDL_WaitThread

Name

SDL_WaitThread -- Espera a la finalización de un hilo de ejecución.

Synopsis

#include "SDL.h"
#include "SDL_thread.h"

void SDL_WaitThread(SDL_Thread *thread, int *status);

Descripción

Espera a la finalización de un hilo de ejecución (no se soportan tiempos máximos de espera).

Valor Devuelto

El valor retornado por la función del hilo se coloca en el área a la que apunta status, sistatus no es NULL.
