SDL_EnableUNICODE

Name

SDL_EnableUNICODE -- Habilita la traducción UNICODE

Synopsis

#include "SDL.h"

int SDL_EnableUNICODE(int enable);

Descripción

Habilita/Deshabilita la traducción UNICODE del teclado.

Si deseas traducir un keysym a su representación imprimible, necesitarás habilitar la traducción UNICODE, usando esta función (enable=0) y consultando el miembro unicode de la estructura SDL_keysym. Este valor será cero para los keysyms que no tengan una representación imprimible. La traducción UNICODE está deshabilitada por defecto debido a que la conversión puede provocar una ligera sobrecarga.

Valor Devuelto

Devuelve el modo de traducción que habí previamente.
