SDL_ActiveEvent

Name

SDL_ActiveEvent -- Estructura de evento de visualización de aplicación

Definición de la estructura

typedef struct{
  Uint8 type;
  Uint8 gain;
  Uint8 state;
} SDL_ActiveEvent;

Datos de la estructura

typeSDL_ACTIVEEVENT.
gain0 si el evento es una pérdida (loss) o 1 si es una obtención (gain).
stateSDL_APPMOUSEFOCUS si se obtuvo o se perdió el foco del ratón. SDL_APPINPUTFOCUS is el foco de la entrada se obtuvo o se perdió, o SDL_APPACTIVE si la aplicación fué iconificada (gain=0) o restaurada (gain=1).

Descripción

SDL_ActiveEvent es un miembro de la unión SDL_Event y se usa cuando se notifica un evento de tipo SDL_ACTIVEEVENT.

Cuando el ratón entra o deja el área de la ventana, se produce un evento de activación de tipo SDL_APPMOUSEFOCUS. Si el ratón entró en la ventana, entonces gain será 1 y en otro caso gain será 0. Un evento de activación de tipo SDL_APPINPUTFOCUS se produce cuando la aplicación pierde u obtiene el foco del teclado. Esto ocurre usualmente cuando otra aplicación se activa. Finalmente, un evento de tipo SDL_APPACTIVE se produce cuando la aplicación o bien se minimiza/iconifica (gain=0), o bien se restaura.

Este evento no se produce cuando la ventana de la aplicación se crea por primera vez.
