SDL_GetRelativeMouseState

Name

SDL_GetRelativeMouseState -- Obtiene el estado actual del ratón.

Synopsis

#include "SDL.h"

Uint8 SDL_GetRelativeMouseState(int *x, int *y);

Descripción

El estado actual de los botones se devuelve como una máscara de bits que puede ser consultada mediante el uso de las macros SDL_BUTTON(X), y los parámetros x e y se establecen al cambio de la posición del ratón desde la última llamada a SDL_GetRelativeMouseState o desde la inicialización del sistema de eventos. Puedes utilizar NULL como valor para x o y.
