Chapter 11. CD-ROM

SDL soporta control de audio de hasta 32 unidades locales de CD-ROM a la vez.

Este API realiza todas las funciones básicas de un reproductor de CDs, incluyendo listado de pistas, reproducción, parada de reproducción, y expulsión del CD-ROM. (Actualmente, no están soportadas unidades de multi-CDs)

Antés de realizar cualquier llamada a las funciones de CD-ROM de SDL, se debe llamar a la función "SDL_Init(SDL_INIT_CDROM)", la cual explora el sistema en busqueda de unidades de CD-ROM, e inicializa el programa para el control de audio. Comprobar el código devuelto, el cual debería ser 0, para ver si hubo algún error al inicializar.

Una vez inicializada la librería, se puede averiguar cuantas unidades hay disponibles usando la función SDL_CDNumDrives(). La primera unidad listada es la unidad de CD-ROM por defecto en el sistema. Una vez elegida una unidad, tras abrirla con SDL_CDOpen(), se puede comprobar su estado y empezar a reproducir si hay un CD en la unidad.

Un CD-ROM esta organizado en una o más pistas, cada una consiste en un numero de "frames". Cada frame es alrededor de 2K de tamaño, y un CD a velocidad de reproducción normal, reproduce 75 frames por segundo. SDL trabaja con el número de frames de un CD, pero estos pueden ser fácilmente convertidos al formato más familiar de minutos/segundos usando la macro FRAMES_TO_MSF().
