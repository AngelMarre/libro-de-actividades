SDL_QuitEvent

Name

SDL_QuitEvent -- Evento de petición de terminación.

Definición de la estructura

typedef struct{
  Uint8 type
} SDL_QuitEvent;

Datos de la estructura

typeSDL_QUIT

Descripción

SDL_QuitEvent es un miembro de la unión SDL_Event y se usa cuando se notifica un evento de tipo SDL_QUIT.

Como se puede observar, la estructura SDL_QuitEvent no tiene un propósito de utilidad. Por otro lado, el evento en sí mismo, es muy importante. Si lo filtras o ignoras un evento de este tipo, entonces será imposible para el usuario cerrar la ventana. De otra manera, si aceptas un evento de tipo terminación(quit), la ventana de la aplicación será cerrada, y las actualizaciones de pantalla seguirán iformando eventos con éxito, aunque la aplicación ya no será visible.

La macro SDL_QuitRequested devolverá un valor distinto de cero si queda algún evento de peticion de terminación pendiente.
