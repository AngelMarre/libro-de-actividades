SDL_KeyboardEvent

Name

SDL_KeyboardEvent -- Estructura de evento para el teclado.

Definición de la estructura

typedef struct{
  Uint8 type;
  Uint8 state;
  SDL_keysym keysym;
} SDL_KeyboardEvent;

Datos de la estructura

typeSDL_KEYDOWN o SDL_KEYUP
stateSDL_PRESSED o SDL_RELEASED
keysymContains key press infomation

Descripción

SDL_KeyboardEvent es un miembro de la unión SDL_Event y se usa cuando se notifica un evento de tipo SDL_KEYDOWN o SDL_KEYUP.

El tipo (type) y estado (state) contienen realmente la misma información, ¡simplemente usan valores distintos para hacerlo! Un evento de teclado se produce cuando se suelta una tecla (type=SDK_KEYUP o state=SDL_RELEASED) y cuando se pulsa una tecla (type=SDL_KEYDOWN o state=SDL_PRESSED). La información que indica que tecla se pulsó o se soltó se almacena en la estructura keysym.

La repetición de eventos de tipo SDL_KEYDOWN se producirá si la repetición de teclas está habilitada (ver SDL_EnableKeyRepeat).
