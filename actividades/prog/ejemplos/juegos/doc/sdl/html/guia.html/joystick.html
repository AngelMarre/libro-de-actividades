Los joysticks(o palancas de juego), y otros dispositivos similares de entrada, tienen un papel muy importante en los juegos y SDL facilita un soporte comprensible para estos. Ejes, botones, minijoysticks de punto de mira(POV Hats) y trackballs están todos soportados.

El soporte para joysticks se inicializa pasando la bandera SDL_INIT_JOYSTICK a la función SDL_Init. Una vez inicializado, los joysticks deben ser abiertos usando la función SDL_JoystickOpen.

Mientras que usando las funciones que se describen en esta sección puede parecer la mejor manera para acceder y consultar los joysticks, en la mayoría de los casos no es así. idealmente los joysticks deberían ser consultados usando el sistema de eventos. Para habilitarlo, debes de activar el estado de procesado de eventos de joystick con la función SDL_JoystickEventState. Los joysticks deben de estar abiertos antes de ser usados, por supuesto.

Si no estas manejando el joystick mediante la cola de eventos, entonces debes hacer una actualizacion de joystick explícita mediante una llamada a la función SDL_JoystickUpdate.

No hay soporte todavia para Force Feedback. Sam (slouken@libsdl.org) está solicitando sugerencias de personas con experiencia en esta característica para encontrar el mejor diseño para la API.
