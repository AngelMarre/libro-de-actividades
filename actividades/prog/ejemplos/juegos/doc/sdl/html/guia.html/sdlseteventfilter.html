SDL_SetEventFilter

Name

SDL_SetEventFilter -- Prepara un filtro para procesar todos los eventos antes de que se manden a la cola de eventos.

Synopsis

#include "SDL.h"

void SDL_SetEventFilter(SDL_EventFilter filter);

Descripción

Esta función prepara un filtro para procesar todos los eventos antes de que se manden a la cola de eventos. Esta es una capacidad muy potente y flexible. El filtro tiene el siguiente prototipo: 
typedef int (*SDL_EventFilter)(const SDL_Event *event);
Si el filtro devuelve 1, entonces el evento será añadido a la cola interna. Si devuelve 0, entonce el evento se eliminará de la cola. Esto permite el filtrado selectivo de forma dinámica.

Existe uns peculiaridad cuando de trata con el evento de tipo SDL_QUITEVENT. El filtro de evento es llamado solo cuando el manejador de ventanas desea cerrar la ventana de aplicación. Si el filtro de evento devuelve 1, entonces la ventana se cerrará, y sinó la ventana permanecerá abierta si es posible. Si el evento de terminación es generado por una señal de interrupción, obviara la cola interna y será despachado a la aplicación en la siguiente comprobación(poll) de eventos.

Los eventos apilados en la cola con SDL_PushEvent o SDL_PeepEvents no pasan por el filtro de eventos.

¡Ten cuidado! La función de filtro de eventos puede ejecutarse en un hilo diferente asi que se cuidadoso con lo que haces en su interior.
