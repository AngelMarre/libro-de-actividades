SDL_DestroySemaphore

Name

SDL_DestroySemaphore -- Destruye un semáforo creado con SDL_CreateSemaphore.

Synopsis

#include "SDL.h"
#include "SDL_thread.h"

void SDL_DestroySemaphore(SDL_sem *sem);

Descripción

SDL_DestroySemaphore destruye el semáforo al que apunta sem y que fue creado mediante SDL_CreateSemaphore. No es seguro destruir un semáforo si aún quedan hilos que están bloqueados esperando a que se abra.

Ejemplos

if (my_sem != NULL) {
        SDL_DestroySemaphore(my_sem);
        my_sem = NULL;
}
