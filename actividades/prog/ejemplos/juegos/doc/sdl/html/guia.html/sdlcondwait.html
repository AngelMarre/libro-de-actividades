SDL_CondWait

Name

SDL_CondWait -- Espera a una variable de condición

Synopsis

#include "SDL.h"
#include "SDL_thread.h"

int SDL_CondWait(SDL_cond *cond, SDL_mutex *mut);

Descripción

Espera a la variable de condición cond y abre el mutex pasado como parámetro. El mutex debe estar bloqueado antes de entrar en esta función. Devuelve 0 cuando recibe la señal, o -1 en caso de error.
