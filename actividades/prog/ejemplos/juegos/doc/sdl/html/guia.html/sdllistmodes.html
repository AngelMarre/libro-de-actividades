SDL_ListModes

Name

SDL_ListModes -- Devuelve un puntero a una matriz de las dimensiones de pantalla disponibles para el formato y flags de video dados.

Synopsis

#include "SDL.h"

SDL_Rect **SDL_ListModes(SDL_PixelFormat *format, Uint32 flags);

Descripción

Devuelve un puntero a una matriz de las dimensiones de pantalla disponibles para el formato y flags de video dados, ordenadas de la más grande a la más pequeña. Devuelve NULL si no hay dimenciones disponibles para un formato particular, o -1 si cualquier dimensión es correcta para el formato dado.

Si format es NULL, la lista de modos será para el formato devuelto por SDL_GetVideoInfo()->vfmt. El parámetro flag es una combinación OR'd de los flags de la superficie. Los flags son los mismos a los usados en SDL_SetVideoMode y juegan un fuerte papel en decidir que modos son válidos. Por Ejemplo, si se pasa SDL_HWSURFACE como flag, solo seran devueltos modos soportados por superficies de video hardware.

Ejemplo:

SDL_Rect **modes;
int i;
.
.
.

/* Obtiene modos disponibles pantallacompleta/hardware */
modes=SDL_ListModes(NULL, SDL_FULLSCREEN|SDL_HWSURFACE);

/* Comprueba si hay algún modo disponible */
if(modes == (SDL_Rect **)0){
  printf("¡ No hay modos disponibles !\n");
  exit(-1);
}

/* Comprueba si la resolución or está restringida */
if(modes == (SDL_Rect **)-1){
  printf("Todas las resoluciones disponibles.\n");
}
else{
  /* Muestra modos válidos */
  printf("Available Modes\n");
  for(i=0;modes[i];++i)
    printf("  %d x %d\n", modes[i]->w, modes[i]->h);
}
.
.
