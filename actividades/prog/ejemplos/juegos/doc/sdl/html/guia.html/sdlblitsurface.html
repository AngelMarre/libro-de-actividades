SDL_BlitSurface

Name

SDL_BlitSurface -- Esto realiza un rápido blit de la superficie origen a la supurficie destino.

Synopsis

#include "SDL.h"

int SDL_BlitSurface(SDL_Surface *src, SDL_Rect *srcrect, SDL_Surface *dst, SDL_Rect *dstrect);

Descripción

Esto realiza un rápido blit de la superficie origen a la supurficie destino.

Solo se usa la posición en dstrect (el ancho y el alto son ignorados)

Si tanto srcrect como dstrect son NULL, se copia la superficie (src or dst entera.

El rectangulo blit es guardado en dstrect después de hacer todos los recortes (clipping) (srcrect no es modificado).

Los resultados de las operaciones (blit) varian ampliamente dependiendo de si SDL_SRCAPLHA está activa o no. Ver SDL_SetAlpha.

La función blit no se debería llamar con una superficie (surface) bloqueada.

Valor Devuelto

Si el blit se realiza correctamente, devuelve 0, en cualquier otro caso, devuelve -1.

Si cualquiera de las superficies estaban en la memoria de video, y el blit devuelve -2, se perdió la memoria de video, asi que se debería re-cargar con su contenido (artwork) y volver a usar el blit: 
        while ( SDL_BlitSurface(image, imgrect, screen, dstrect) == -2 ) {
                while ( SDL_LockSurface(image)) < 0 )
                        Sleep(10);
                -- Write image pixels to image->pixels --
                SDL_UnlockSurface(image);
        }
Esto ocurre bajo DirectX 5.0 cuando el sistema cambia/sale de pantalla completa. Bloqueando la superficie también fallará hasta que se tenga acceso a la memoria de video de nuevo.
