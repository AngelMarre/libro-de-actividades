SDL_JoyHatEvent

Name

SDL_JoyHatEvent -- Joystick hat position change event structure

Definición de la estructura

typedef struct{
  Uint8 type;
  Uint8 which;
  Uint8 hat;
  Uint8 value;
} SDL_JoyHatEvent;

Datos de la estructura

typeSDL_JOY
whichÍndice del dispositivo del joystick
hatÍndice del minijoystick(hat) del joystick
valuePosición del minijoystick(hat)

Descripción

SDL_JoyHatEvent es un miembro de la unión SDL_Event y se usa cuando se notifica un evento de tipo SDL_JOYHATMOTION.

Un eventeo de typo SDL_JOYHATMOTION se produce siempre que el usuario mueve un minijoystick del joystick. El campo which es el índice del joystick que produjo el evento y el campo hat es el índice del minijoystick (para una explicación más detallada ver la sección de Joystick). El campo value es la posición actual del minijoystick. Es un OR lógico de los siguientes valores (cuyos significados deberían ser bastante obvios.:) :

SDL_HAT_CENTERED
SDL_HAT_UP
SDL_HAT_RIGHT
SDL_HAT_DOWN
SDL_HAT_LEFT

Las siguientes definiciones también se facilitan:

SDL_HAT_RIGHTUP
SDL_HAT_RIGHTDOWN
SDL_HAT_LEFTUP
SDL_HAT_LEFTDOWN
