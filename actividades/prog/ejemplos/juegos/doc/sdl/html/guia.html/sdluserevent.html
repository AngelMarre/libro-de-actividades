SDL_UserEvent

Name

SDL_UserEvent -- Tipo de evento definido por el usuario.

Definición de la estructura

typedef struct{
  Uint8 type;
  int code;
  void *data1;
  void *data2;
} SDL_UserEvent;

Datos de la estructura

typeSDL_USEREVENT hasta SDL_NUMEVENTS-1
codeCódigo de evento definido por el usuario
data1Puntero de datos definido por el usuario
data2Puntero de datos definido por el usuario

Descripción

SDL_UserEvent se encuentra en el miembro user de la estructura SDL_Event. Este evento es único, nunca es creado por la librería SDL, solo por el usuario. El evento puede ser apilado en la cola de eventos usando SDL_PushEvent. el contenido de los miembros de la estructura son cuestión únicamente del programador, siendo el único requerimiento que el campo type sea un valor comprendido entre SDL_USEREVENT y SDL_NUMEVENTS-1 (ambos inclusive).

Ejemplos

SDL_Event event;

event.type = SDL_USEREVENT;
event.user.code = my_event_code;
event.user.data1 = significant_data;
event.user.data2 = 0;
SDL_PushEvent(&event);
