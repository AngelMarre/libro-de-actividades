Herramientas

  • J2SE
    • Compilador java.
    • Librerias, API java.
    • Máquina virtual java.
    • Soporte java.
    • Distribuidores: Sun, IBM, etc.
  • ANT
    • Facilita la gestión de proyectos.
    • Requiere soporte java.
    • Distribuidores: Proyecto Apache
    • Ver manual de ANT
  • TOMCAT
    • Motor de páginas JSP y Servlet.
    • Require soporte java.
    • Distribuidores: Proyecto Apache

Instalaciones

  • J2SE (Java 2 Standard Edition versión 1.3.2)
    1. Descargar ficheros de la página web de Sun y descomprimir (Ir a http://java.sun.com).
    2. Definir variable de entorno JAVA_HOME. Actualizar PATH.
    3. Comprobar (java -version).
  • ANT (Versión 1.6.1)
    1. Descargar ficheros de la página web de Apache y descomprimir (Ir a http://www.apache.org).
    2. Definir variable de entorno ANT_HOME. Actualizar PATH.
    3. Comprobar (ant -version).
  • TOMCAT (Versión 5.0.19)
    1. Descargar ficheros de la página web de Apache y descomprimir (Ir a http:/www.apache.org).
    2. Definir variable de entorno CATALINA_HOME. Actualizar PATH.
    3. Iniciar: catalina start. Parar: catalina stop.
    4. Comprobar http://localhost:8080/
