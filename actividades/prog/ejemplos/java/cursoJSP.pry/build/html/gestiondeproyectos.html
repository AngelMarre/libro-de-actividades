Gestión de Proyectos

Tomcat: Creación de un usuario de administración

  1. Modificar el fichero tomcat-users.xml para añadir un nuevo usuario. Incluir la siguiente línea:
    <user username="nombredeusuario" password="tomcat" roles="manager,admin,tomcat,role1">
  2. Reiniciar el servidor TOMCAT

Tomcat: Consultar opciones de administración

  1. Status: Consultar el estado del contenedor.
  2. Manager: Recargar, Replegar y Desplegar aplicaciones WEB.
    3. /manager
  3. Administrador
    4. /admin

ANT: Configurar administración de Tomcat

  1. Copiar el fichero CATALINA_HOME/server/lib/catalina-ant.jar en ANT_HOME/lib.
  2. Modificar el fichero build.xml
  3. Comprobar que las líneas taskdesf no están comentadas.
  4. Crear las property manager.username y manager.password.
    5.         <property name="manager.username"   value="nombredeusuario"/>
        <property name="manager.password"   value="clavedeusuario"/>

ANT: Instalar proyectos WEB

Preparar el proyecto:

  1. $ ant clean
  2. $ ant all
  3. $ ant dist

Instalar en TOMCAT:

  • $ ant install

Listas las aplicaciones WEB:

  • $ ant list
