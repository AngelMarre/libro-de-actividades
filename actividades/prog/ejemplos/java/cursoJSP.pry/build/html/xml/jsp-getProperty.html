<jsp:getProperty>

Leer el valor de la propiedad del bean.

Sintaxis JSP

<jsp:getProperty name="beanInstanceName" property="propertyName" />

Sintaxis XML

<jsp:getProperty name="beanInstanceName" property="propertyName" />

Ejemplos

<jsp:useBean id="calendar" scope="page" class="employee.Calendar" />	
<h2>	
Calendar of <jsp:getProperty name="calendar" property="username" />	
</h2>

Descripción

La etiqueta <jsp:getProperty> lee el valor de la propiedad del bean y añade al valor a la respuesta. Declarar el bean (<jsp:useBean>) antes de usarlo con <jsp:getProperty>.

<jsp:getProperty> tiene algunas limitaciones:

  • Se pueden recuperar valores indexados.
  • Se puede usar con JavaBeans pero no con "enterprise java beans".

    • Atributos

    • name="beanInstanceName"
    • property="propertyName"

      Nombre de la propiedad del bean cuyo valor se quiere leer. La propiedad se declara como una variable en el bean y debe tener un método SET asociado (Para más información consultarhttp://java.sun.com/products/javabeans/docs/).

    Recordar

    Si se usa <jsp:getProperty> para recuperar un valor nulo se genera una excepción NullPointerException. Sin embargo, si se usa un scriptlet o expresion para recuperar el valor tenemos un tipo string con valor null que se mostrará en el navegador.
