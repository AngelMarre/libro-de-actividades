Mi Primer Proyecto

  • Conocer TOMCAT
    • Directorio base CATALINA_HOME=../jakarta-tomcat-5.0.19
    • bin: Ficheros ejecutables.
      • Arrancar: catalina start
      • Parar: catalina stop.
    • common: Ficheros comunes.
    • conf: Ficheros de configuración.
    • logs: Ficheros de log.
    • server:.
    • shared:.
    • temp: Ficheros temporales.
    • webapps: Aplicaciones web.
    • work:.
  • Mi proyecto
    • Copiar directorio sample de CATALINA_HOME/webapps/tomcat-docs/appdev/sample
    • Renombrar directorio para mi proyecto.
    • Directorio web:
      • images: ficheros gif
      • WEB-INF: fichero web.xml, directorios classes y lib
      • hello.jsp
      • index.html
