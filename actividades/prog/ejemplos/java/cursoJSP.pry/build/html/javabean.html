JavaBean

Uso de los JavaBean con JSP

  • Características/necesidades mínimas
    1. Tener un constructor sin argumentos (vacío).
    2. No contar con variables de instancia públicas (campos).
    3. Acceder a valores persistentes a través de métodos llamados getXXX(o isXXX) y setXXX.
  • Uso básico de los Beans
    • <jsp:useBean id="nombre" class="paquete.Clase" />
    • <jsp:getProperty name="nombre" property="propiedad"/>
    • <jsp:setProperty name="nombre" property="propiedad" value="valor"/>
    • El atributo value puede tomar una expresión JSP.
  • Atributo scope
    • page: El bean se coloca en el objeto PageContext.
    • application: El bean se almacena en SevletContext.
    • session: El bean se almacena en HttpSession.
    • requets: El bean se coloca en el objeto ServletRequets.
