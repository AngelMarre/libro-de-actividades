Directiva Include

Incluye un fichero estático en la página JSP.

Sintaxis JSP

<%@ include file="URLrelativa" %>

Sintaxis XML

<jsp:directive.include file="URLrelativa" />

Ejemplos

include.jsp:
<html>	
<head><title>Ejemplo de include</title></head>	
<body bgcolor="white">	
<font color="blue">	
Fecha y hora actual	
<%@ include file="fechahora.jsp" %>	
</font>	
</body>	
</html>

fechahora.jsp:
<%@ page import="java.util.*" %>	
<%= (new java.util.Date() ).toLocaleString() %>
Muestra en la página:
Fecha y hora actual	
Aug 30, 1999 2:38:40

Descripción

La directiva include directive inserta un fichero de texto o con código en una página JSP en el momento de realizar la compilación.

Hay que tener en cuenta que:

  • El fichero insertado puede ser otra página JSP, una página HTML, un documento XML, un fichero de texto, etc.
  • Las etiquetas del fichero insertado pueden entrar en conflicto con la estructura del fichero principal provocando errores. Por ejemplo: <html>, </html>, <body>, o </body>.
  • La página JSP puede ser recompilada si el fichero incluido cambia.
  • Existe un include estático de la forma <jsp:include page="URL relativo">

Atributos

file="URLrelativa"

La ruta del fichero incluido siempre será un URL relativo. URL relativo es una ruta sin el protocolo, puerto o nombre de dominio, como por ejemplo:

"error.jsp"	
"/templates/onlinestore.html"	
"/beans/calendar.jsp"

Si la URL empieza con /, la ruta es relativa al contexto de la aplicación JSP.

Si la URL empieza con un nombre de directorio o de fichero, la ruta es relativa a la página JPS.
