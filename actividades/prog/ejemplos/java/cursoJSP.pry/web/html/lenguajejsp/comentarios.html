Comentarios

Código de las páginas JSP que no se añade a la respuesta.

Sintaxis JSP

<%-- comentarios --%>

Sintaxis XML

Ninguna

Ejemplo

<%@ page language="java" %>	
<html>	
<head><title>Ejemplo de comentarios</title></head>	
<body>	
<h2>Ejemplo de comentarios</h2>	
<%-- Esta línea no se incluye en la respuesta --%>	
</body>	
</html>

Descripción

El texto entre las marcas de comentarios es ignorado por el contenedor JSP. El contenedor JSP no procesa nada contenido entre las marcas <%-- y --%>. Los comentarios no se añaden por tanto a la respuesta.

No confundir los comentarios JSP con los comentarios HTML (<!-- Comentarios HTML -->)
