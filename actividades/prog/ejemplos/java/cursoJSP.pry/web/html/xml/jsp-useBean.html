<jsp:useBean>

Localiza e inicializa un bean con un nombre y un ámbito.

Sintaxis JSP

<jsp:useBean id="beanInstanceName"	
   scope="page|request|session|application"	
{	
   class="package.class" [ type="package.class" ]|	
   beanName="{package.class | <%= expression %>}" 	
      type="package.class" | 	
   type="package.class"	
}	
{ /> | > other elements </jsp:useBean> }

Sintaxis XML

<jsp:useBean id="beanInstanceName"	
   scope="page|request|session|application"	
{	
   class="package.class"  [ type="package.class" ]  |	
   beanName="{package.class | %= expression %}"	
      type="package.class" |	
   type="package.class"	
}	
{ /> | > other elements </jsp:useBean>  }

Ejemplos

<jsp:useBean id="cart" scope="session" class="session.Carts" />	
<jsp:setProperty name="cart" property="*" />
<jsp:useBean id="checking" scope="session" class="bank.Checking" >	
   <jsp:setProperty name="checking" property="balance" value="0.0" />	
</jsp:useBean>

Descripción

La etiqueta <jsp:useBean> localiza e inicializa un componente JavaBeans. <jsp:useBean> primero intenta localizar el bean. Si no existe, <jsp:useBean> lo instancia desde una clase.

Pasos para localizar e instanciar el Bean:

  1. Se intenta localizar el bean con el ámbito y nombre especificado.
  2. Se define una variable que referencia al objeto con el nombre especificado.
  3. Si se encuentra el bean, se almacena la referencia en la variable.
  4. Si no se encuentra el bean, se instancia de la clase especificada, se almacena la referencia en una nueva variable (java.beans.Beans.instantiate).
  5. Si <jsp:useBean> está instanciado y tiene elementos en el cuerpo entonces se ejecutan.

El cuerpo de un <jsp:useBean> contiene elementos <jsp:setProperty>. La estiquetas del cuerpo se procesan si <jsp:useBean> instancia el bean. Si ya existe las etiquetas no tienen efecto.

Se puede usar <jsp:useBean> para trabajar con JavaBeans pero no con Enterprise Java Bean.

Atributos

  • id="beanInstanceName"

    Variable que identifica el bean dentro de su ámbito.

  • scope="page|request|session|application"

    Ámbito dentro del cuál existe el bean y la variable id está disponible. El valor por defecto es page. Opciones:

    page   Dentro de la página JSP.

    request   Desde cualquier página JSP que procese la misma petición. Se puede usar el objeto request para acceder al bean, por ejemplo, request.getAttribute(beanInstanceName).

    session   Se puede usar el bean dentro de la misma sesión. La página que crea el bean debe tener la directiva page con el atributo session="true".

    application   Se puede usar el bean dentro de la misma aplicación. Cualquier página de la aplicación puede usar el bean.

  • class="package.class"

    Clase desde donde se instancia. La clase no puede ser abstracta, debe ser pública y sin argumentos.

  • type="package.class"
  • class="package.class" type="package.class"
  • beanName="{package.class | <%= expression %>}" type="package.class"
