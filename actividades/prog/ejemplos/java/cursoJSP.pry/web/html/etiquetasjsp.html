Introducción

  • JSP compuestas por varios tags o etiquetas. Ejemplos de etiquetas JSP son setProperty, include, forward, usebean.
  • Todas encapsulan comportamientos de una manera sencilla.
  • La etiqueta crea una instancia de una clase definida como un JavaBean. El uso de la etiqueta en conjunción con los JavaBeans hace que podamos separar la lógica de presentación de la de proceso y consigue unos archivos JSP mas claros, y lo que es mas importante, el que escribe la JSP, no tiene por que tener unos conocimientos avanzados de programación para crear páginas JSP de funcionalidades avanzadas.
  • Las JSP Custom Tags son un paso mas allá, con ellas se pueden añadir nuevas etiquetas a las ya existentes.

Etiquetas personalizadas

  1. Se implementa una clase Java que hace de manejador de etiquetas.
    /WEB-INF/classes/*.class
  2. Se define un descriptor de librería de etiquetas (TDL).
    /WEB-INF/tld/*.tld
  3. Se mapea librería con el descriptor.
    /WEB-INF/web.xml
  4. Se crear página JSP con las etiquetas personalizadas.

Tipos

  • Hay dos tipos principales de etiquetas, etiquetas que no tienen cuerpo y etiquetas que si lo tienen.
  • Ejemplos son la etiqueta <img src="asdfa.gif"> que no tiene cuerpo y no tiene etiqueta de final, y <title>manolo</title> que tiene cuerpo y etiqueta de final.
