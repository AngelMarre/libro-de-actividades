Mi Segundo Proyecto

  • Conocer ANT
    • Directorio base ANT_HOME=../apache-ant-1.6.1
    • bin: Ficheros ejecutables.
      • Arrancar: catalina start
      • Parar: catalina stop.
    • common: Ficheros comunes.
    • conf: Ficheros de configuración.
    • logs: Ficheros de log.
    • server:.
    • shared:.
    • temp: Ficheros temporales.
    • webapps: Aplicaciones web.
    • work:.
  • Fichero build.xml
    • $ ant (Target por defecto)
    • $ ant compile (Genera directorio build)
    • $ ant clean (Elimina el directorio build)
    • Ver build.xml de ejemplo
    • taskdef y catalina-ant.jar
    • Definir la variable catalina.home
  • Mi proyecto
    • docs: documentación del proyecto, javadoc, etc.
    • src: contiene los fuentes java, paquetes, javabean, etc.
    • web: contiene los ficheros html, jsp, gif, etc.
      • html: ficheros html
      • images: ficheros gif
      • jsp: ficheros jsp
      • pdf: ficheros pdf
      • txt: ficheros txt
      • WEB-INF: fichero web.xml, directorios classes y lib
      • xml: ficheros xml
