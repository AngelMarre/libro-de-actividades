Barraquito (Proyecto TENERIFE)

Un poco de historia

El proyecto TENERIFE conocido cariñosamente como BARRAQUITO nace a finales de 2004 cuando me encuentro embarcado en un proyecto de desarrollo para una empresa privada. Claro está que no era mi primer proyecto y por eso me surgieron varias preguntas:

  • ¿Por qué motivo con cada nuevo proyecto de desarrollo de software me veo repitiendo una y otra vez el mismo patrón de programación?
  • ¿Por qué no ahorrarme todo ese esfuerzo?
  • Si existiera una forma de aprovechar mi tiempo ¿podría asumir proyectos más complejos y costosos?
  • ¿Podría llegar un tiempo que un programador pueda sustituir a un gran equipo de trabajo?
  • ¿Podría hacerse todo esto y además mantener una gran calidad de producto final?
  • y lo más importante ¿me podría entretener haciendo algo que me divierta mientras programo y aprendo con java?

Todavía no tengo la respuesta a estas preguntas, pero lo que si tengo es un camino por recorrer y además un reto divertido que afrontar.

Así fue como comenzó todo.

Introducción

BARRAQUITO es una aplicación para la creación de programas. El nombre BARRAQUITO es un homenaje a java y a esa bebida típica de Tenerife a base de café, leche, leche condensada... bueno sigamos con el tema.

Es cierto que ya existen otros FrameWorks que realizan la misma tarea y que los conceptos utilizados en BARRAQUITO no son nada novedosos. Pero aunque la mayoría de los FrameWorks generan un prototipo, luego es necesario que un programador lo termine escribiendo unas pocas líneas y quede finalmente operativo.

La idea de BARRAQUITO es la de generar programas que cumpliendo tres requisitos

  1. Sencillez. Ante todo no queremos perder tiempo aprendiendo cómo funciona el último FrameWork de moda. Sólo queremos usarlo.
  2. Funciona. Esto es, sin requerir ninguna modificación por parte del programador la aplicación ya está funcionando.
  3. Escalable. Es decir, si queremos refinar y mejorar la aplicación que se ha generado podemos hacerlo puesto que partimos de un prototipo. Con la ayuda de un programador podemos retocar/reprogramar el proyecto para adaptarlo a nuestras necesidades particulares.

Actualmente BARRAQUITO está en la versión 0.1 y ya es capaz de generar aplicaciones funcionales para los siguientes lenguajes:

  • Java (swing)
  • Java Server Pages
  • PHP
  • SQL (PostGreSQL)

En la próxima versión se espera poder generar además proyectos para:

  • Java (swt)
  • Java Server Faces
  • PHP
  • SQL (MySQL)
