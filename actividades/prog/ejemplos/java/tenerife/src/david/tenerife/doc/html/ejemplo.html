Definición

Para establecer las características del proyecto que queremos construir debemos crearnos un fichero de definición del mismo.

Actualmente en la versión 0.1 de BARRAQUITO la creación de este fichero se realiza de forma manual. Aún así el sistema es muy sencillo. Está previsto para la próxima versión la creación de una herramienta visual que nos facilitará todavía más esta tarea.

Ejemplo del fichero

basedatos consultas paqueteJava=proyecto nombreProyecto=Nombre_Proyecto
tabla personas
campo cod_persona integer clave etiqueta=CODIGO
campo nombre string descripcion
campo apellido1 string descripcion
campo apellido2 string descripcion
campo fecha_nacimiento date
campo dni string expresionRegular=asterisco
campo telefono1 string
campo telefono2 string
campo direccion string

tabla agendas
campo cod_agenda integer clave etiqueta=CODIGO
campo des_agenda string descripcion etiqueta=DESCRIPCION
