# Estructura secuencial (I)

Ejemplo 1
```
# Sumar enteros
a = 3
b = 4

puts a + b
# 7
```

Ejemplo 2
```
# Unir Strings
a = “Hello”
b = “World!”
puts a + b

# “HelloWorld”
```

**Rule1:** _Declaración de variables dinámica._

Ejemplo 3
```
# Sumar enteros con Strings
a = “1”
b = 0

puts a + b

# ¿10 o “10”?
# Error: No implicit conversion of
         Integer into String
```

**Rule2:** _Tipado fuerte._
