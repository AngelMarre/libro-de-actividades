# Estructura iterativa: los bucles (I)

Ejemplo 1:
```
i = 0
while (i<10)
  puts i
  i++
end

# <i> es un objeto de tipo Integer
```

Ejemplo 2:

```
for i in 0..9
  puts i
end

# <0..9> es un objeto de tipo Range
```
